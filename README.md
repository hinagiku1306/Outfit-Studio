# Outfit Studio

An in-game wardrobe menu for [Stardew Valley](https://www.stardewvalley.net/) to browse, customize, save, and organize outfits.

Built for **Stardew Valley 1.6+** using [SMAPI](https://smapi.io/).

## Features

- **Outfit Menu** — View and change your farmer's clothings and hairstyles from an in-game UI
- **Wardrobe** — Save outfits as named sets, tag them, and filter/search your collection
- **Schedule System** — Automatically switch outfits based on configurable rules

## Requirements

- [Stardew Valley](https://www.stardewvalley.net/) 1.6.0+
- [SMAPI](https://smapi.io/) 4.0.0+
- [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) (optional, for config UI outside the game menu)

## Installation

1. Install SMAPI
2. Download the latest release from [Nexus Mods](https://www.nexusmods.com/stardewvalley/mods/42090) or chosen version under Releases
3. Extract the `OutfitStudio` folder into your `Stardew Valley/Mods/` directory
4. Launch the game through SMAPI

## Default Keybindings

| Key | Action |
|-----|--------|
| `O` | Toggle Outfit Studio menu |

Keybindings are customizable in the mod config.

## Configuration

All settings can be changed in-game through the built-in config menu or via Generic Mod Config Menu. Options include:

- **Keybindings** for menus and item info
- **UI behavior** — close on outside click, tooltip visibility, auto-focus search, arrow key scrolling
- **Wardrobe defaults** — search scope, filter reset behavior, visible rows/columns
- **Schedule settings** — priority, warp behavior, debug logging
